# ADR 0001 - Estrutura Inicial do Projeto

## Contexto
O projeto CentavoControl segue DDD, Clean Architecture e práticas modernas.

## Problema
Definir estrutura inicial alinhada ao AGENTE.md.

## Decisão
Estruturar pastas: Domain, Application, Infrastructure, WebApi, tests, docs.

## Consequências
Facilita manutenção, testes e evolução.
